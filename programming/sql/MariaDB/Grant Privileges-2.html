GRANT ALL PRIVILEGES ON *.* TO 'username'@localhost IDENTIFIED BY 'password';

The *.* refers to the database or table for which the user is given privileges.  Using *.* grants access to all databases located on the server.  This could be a major security issue.  It is advised to grant privileges to the specific database the user will work with.

To grant privileges only for yourDB, use the following statement:

GRANT ALL PRIVILEGES ON 'yourDB'.* TO 'username'@localhost;

Be sure to refresh privileges once new ones have been awarded

FLUSH PRIVILEGES;
