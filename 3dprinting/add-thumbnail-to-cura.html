Add Thumbnail to Cura

Contents

To enable Cura to automatically generate a thumbnail image with each project, go to Extensions>Post Processing>Modify G-Code and click Add a Script. Enter a height and width value (I’ve standardized on 300x300). This setting will remain in effect until it is changed or removed.

You can also add a second thumbnail size (32x32 or 48x48) for smaller file thumbnails.
