Purpose

Provides  Python based configuration of the i3wm bar.

Prerequisites

  • i3wm
  • i3bar
  • Python3

Procedure

Add py3status to i3bar Config

Change bar portion of the i3wm config file as follows

status_command py3status

This will use the default i3status config file for layout of the bar.  Since there can be multiple bar instances within the i3wm config file, the user can specify a specific i3status config file for each bar.  An alternate config file can be used as follows

status_command py3status -c {path-to-config}
