Save Volume Settings

Purpose

Initial max volume on Debian system is often quite below an acceptable level.  This procedure will allow the user to adjust audio input and output to desired levels and save configuration in the user’s home directory. 

Prerequisites

  • Superuser privileges
  • alsamixer installed
  • alsactl installed

Procedure

Adjust Audio

Start alsamixer and change settings to desired levels

alsamixer

Make adjustments to each master channel on the initial screen.  If additional audio settings are to be changed, press F5 to see all available settings.

Once all adjustments are as desired, press Esc to exit alsamixer

Save Configuration

The following will save alsamixer configuration to the user’s in the .config directory within the user’s home directory.

sudo alsactl --file ~/.config/asound.state store

Reload Configuration

If it is necessary to reload alsamixer configuration, use the following command

sudo alsactl --file ~/.config/asound.state restore
