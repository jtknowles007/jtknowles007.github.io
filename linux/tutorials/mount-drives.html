Mount Drives

Prerequisites

  • Superuser privileges

Procedure

List Partitions

Check the disk partitions already created on your system that are not already mounted using the following command:

lsblk -f

Create mount point

Mount points can be placed in one of two locations within the system, or anywhere desired in the user’s home directory.  It is common to mount permanent drives in /mnt and temporary drives or other removable media, such as flash drives, in /media.  There is no functional difference between the two locations and the user is free to mount drives anywhere they see fit.  The mount point can be created using the following command:

sudo mkdir /{path-to-mountpoint}/{mountpoint}

NOTE: The use of sudo in the above example is not necessary if the mount point will be located within the user’s home directory. ### Mount Drive Mount the desired partition using the following command:

sudo mount /dev/{partition} {path-to-mountpoint}/{mountpoint}
