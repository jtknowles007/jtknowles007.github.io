Extend LVM Partition

Logical Volume Manager (LVM) provides the means to increase and reduce file system size on the fly without downtime. If we have a Volume Group partition running out of space and want to extend or increase the size of the filesystem, one can do so as long as there is free space available in the volume group.

Prerequisites

  • Root privileges
  • Live CD on USB Drive
  • df
  • lvm tools

Procedure

View Disk Usage of File System

Run df to view total, used, and available disk space

df -h

Check Free Space in Volume Group

To display Volume Group details, run vgdisplay followed by volume group name

vgdisplay <volume-group-name>

Boot into Live CD from USB Drive

The Volume Group that will be acted upon must be unmounted to make changes. In order to do this, boot into a Live CD to conduct the majority of this procedure.
