Create Symlink

A symlink, or symbolic link, is a special type of file that points to another file or directory.  This is similar to a shortcut in Windows

Prerequisites

  • ln
  • rm

Procedure

Use the following command to create a symlink.  

ln -s {target file} {link name}

The -s option is important to create a soft link instead of a hard link

To delete a symlink you can just remove the link using:

rm {link name}

This will delete the link, not the original file.

Things to keep in mind:

  • Changes made to the symlink are reflected in the original file
  • You can chain symbolic links
