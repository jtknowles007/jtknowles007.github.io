Purpose

A resource for many (not all) Vim commands to make document editing more enjoyable. # Prerequisites * Vim # Syntax ## Insert & Replace | Command | Action | |—–|—–| | i | insert text before the cursor | | a | insert text after the cursor | | I | insert text at the beginning of the current line | | A | insert text at the end of the current line | | o | insert text in a new line below the cursor | | O | insert text in a new line above the cursor | | r | replace a character at the cursor position | | R | replace characters starting at the cursor position | ## Cursor Motion | Command | Action | |—–|—–| | h | insert text before the cursor | | l | insert text after the cursor | | k | insert text at the beginning of the current line | | j | insert text at the end of the current line | | G | goto the end of file | |nG or :n| goto the line number n | | 0 | move to the beginning of the line | | $ | move to the end of the line | | CTRL+f | move one screen view forward | | CTRL+b | move one screen view backward | ## Ex Commands | Command | Action | |—–|—–| | :q | quit current open file | | :q! | force quit without saving changes | | :w | save file | | :w {filename} | save file as {filename} | | :wq! | force save then quit | | :w,yw {filename} | wrate from line x to line y into {filename} | | :w >> {filename} | append buffer to {filename} | | :e {filename} | edit another file | # References 1.
