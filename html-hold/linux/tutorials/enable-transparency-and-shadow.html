Enable Transparency and Shading

There is no built in compositing in i3wm (and many other desktop environments).  In order to enable features such as transparency, blurs, and shadows, the Compton compositor is one of the most popular compositors for Linux.

Prerequisites

  • Superuser privileges
  • Compton installed

Procedure

Configure Compton

Compton looks for a config file in many different places, but the most convenient location is ~/.config/compton.  If this directory does not exist, go ahead and create it before moving forward.

Create a file named compton.conf within the compton directory.  All the rules that you want compton to follow will go into this file.
