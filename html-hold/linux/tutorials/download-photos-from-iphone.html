Download Photos from iPhone

Prerequisites

  • ifuse
  • Rapid Photo Downloader
  • File browser of your choice

Procedure

Connect iPhone to computer.  Be sure to unlock the phone once connected and select “trust” if asked to do so.

launch ifuse from the command line using the following:

ifuse ~/AppleIphone && cd ~/AppleIphone

Launch Rapid Photo Downloader and wait for photos and videos to be read into the program

Select the directory you would like the photos and videos to be saved into

Click the download button and wait for the files to complete download

Start file browser and confirm that the files are downloaded

Close Rapid Photo Downloader

Unmount iPhone from system by executing the following command:

fusermount -u ~/AppleIphone

Disconnect iPhone from computer and delete photos from the phone to free up space

References
