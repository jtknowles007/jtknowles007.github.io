Create Bootable USB from ISO

Bootable Linux distributions are available and can be written to a USB drive instead of a traditional CD/DVD. The ISO can be easily written to the USB drive using the command line.

Prerequisites

  • USB Drive of sufficient size to accomodate ISO
  • ISO file
  • Root privileges
  • dd command

Procedure

Obtain ISO

Download ISO from the appropriate repository and save to the location of your choice. On this system it is ~/ISO/

Setup USB Drive

Make sure USB drive is plugged in then open terminal and use the following command determine the label of the USB drive.

sudo fdisk -l

Take note of the label, which should be similar to /dev/sdX

Write ISO to USB Drive

Use the following dd command to make a bootable USB drive from the ISO file

sudo dd bs=4M if=/path/to/file.iso of=/dev/sdX status=progress

Where /path/to/file.iso is the actual path to the ISO file, and /dev/sdX is the label of the USB drive noted previously. dd will copy the contents of the ISO file over to the USB drive. When it is done, the USB drive is ready to use.
