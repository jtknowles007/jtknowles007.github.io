Purpose

Locations within the user’s home directory for various files, applications, and other files not frequently accessed.

Documents

Resource Location
Local Documents ~/Documents/
Synchronized Documents ~/Dropbox/

Media Files

Resource Location
Photos ~/Pictures/
Music ~/Music/
Disc Images ~/ISO/
Videos ~/Videos/
Mame Roms ~/mame/roms/

Programs and Scripts

Resource Location
User Scripts ~/bin/
User Applications ~/.local/bin/
User Software Projects ~/Projects/
Blender Projects ~/Blender/
Pycharm Projects ~/PycharmProjects/

Configuration Files

Configuration files, often called dotfiles, are managed by the stow application.  Stow allows for the dotfiles to be kept in one directory tree and creates symlinks in the appropriate place pointing to the files within the Dotfiles directory.  The files are all synchronized to GitHub to ensure they are preserved and can be shared with other interested individuals.

Resource Location
Xresources ~/Dotfiles/xresources/
i3wm ~/Dotfiles/i3/.config/i3/
i3status ~/Dotfiles/i3status/.config/i3status/
rofi ~/Dotfiles/rofi/.config/rofi/
bash ~/Dotfiles/bash/
vim ~/Dotfiles/vim/
